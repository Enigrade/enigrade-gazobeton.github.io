    Сфера применения газобетона
    Наиболее высокое значение удельной эффективной активности радия — 226 имеют глины, щебень из гранита, керамзитовый гравий. При изготовлении в заводских условиях керамзитового гравия из бентонитовых глин с высоким содержанием радионуклидов возможно образование технологических зон, опасных для здоровья персонала. Сам керамзит, широко применяемый в жилищном строительстве, по мнению многих специалистов, представляет потенциальную угрозу для здоровья человека.
